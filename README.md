Canvas Espresso Narrative ☕
A premium, interactive scrollytelling web application built to showcase an espresso narrative. This project utilizes an HTML5 Canvas sequence synchronized perfectly with user scroll, offering a cinematic and highly immersive front-end experience.

✨ Key Features
Interactive Scrollytelling: Scroll-bound canvas animations that provide a seamless narrative flow.
Premium UI/UX: Luxury aesthetic with minimalist design, powered by modern React and Framer Motion.
Full-Stack Prepared: Built on TanStack Start, allowing for powerful Server-Side Rendering (SSR) and seamless server functions integration using Nitro.
Accessible Components: Accessible and beautifully designed components using Radix UI.
Responsive Architecture: Fully mobilized responsive layout using Tailwind CSS v4.
Production Ready: Pre-configured for seamless deployments onto edge networks like Netlify.
🛠️ Technology Stack
Framework: React 19 + Vite
Routing & SSR: TanStack Start & TanStack Router
Styling: Tailwind CSS v4
State Management & Data Fetching: TanStack Query
Animations: Framer Motion
UI Primitives: Radix UI
Icons: Lucide React
🚀 Getting Started
Prerequisites
Ensure you have Node.js and npm (or bun/yarn) installed on your system.

Installation
Clone the repository:
git clone https://github.com/Jithin-UN/cafeteria.git
Navigate into the project folder:
cd cafeteria/canvas-espresso-narrative-main
Install dependencies:
npm install
Development Server
To start the local development server:

npm run dev
The application will be available at http://localhost:8080.

Production Build
To build the application for production, run:

npm run build
This will run the deployment pipeline and generate the production files in the dist and .netlify directories.

☁️ Deployment (Netlify)
This project comes pre-configured with a netlify.toml file utilizing the Nitro engine preset to easily deploy onto Netlify's Edge network.

Create a project in your Netlify Dashboard and import this GitHub repository.
Netlify will automatically detect the build settings via the netlify.toml file.
Click "Deploy Site" to instantly compile and roll out the live production version.
